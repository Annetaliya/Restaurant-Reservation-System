# Restaurant Reservation System
This projects demontrates a booking system where a user is a able to reserve a table at a certain date. Features include notifications to admin when a booking is done.
 
## Built With the following

- CSS
- React.js
- Express Js
- postgresql
- supabse
- JSX
- Bootstrap
- Linters
